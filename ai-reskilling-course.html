05

Curriculum

Module 1: Essential AI Theory for Practitioners

3 days (9 hours) 4 live + 5 self-paced

Core Topics

  • LLM lifecycle: pre-training → fine-tuning → inference
  • RAG fundamentals: retrieval + generation patterns
  • Agent architecture: model + tools + memory + output
  • Multi-agent patterns and orchestration

Hands-on Labs

  • Run 7B LLM and examine token probabilities
  • Query vector database with live documents
  • Build simple Pydantic AI agent with tools

Module 2: Building RAG Applications

1.5 weeks (24 hours) 9 live + 15 self-paced

Core Topics

  • RAG pipeline: Load → Chunk → Embed → Index → Retrieve → Generate
  • Embeddings and vector search with FAISS
  • Context assembly and prompt engineering
  • API development with FastAPI and evaluation metrics

Hands-on Labs

  • Build document ingestion pipeline for 20 PDFs
  • Implement metadata filtering (dept, date ranges)
  • Create RAG API with citation tracking
  • Deploy to Render/Railway with production testing

Module 3: Fine-Tuning LLMs

1 week (16 hours) 6 live + 10 self-paced

Core Topics

  • IBM Granite 8B: enterprise-safe, Apache 2.0 licensed
  • Efficient fine-tuning with LoRA and QLoRA (4-bit)
  • Inference server deployment with vLLM
  • Model quantization for production environments

Hands-on Labs

  • Fine-tune Granite 8B on 500 support tickets
  • Benchmark base vs. fine-tuned performance
  • Deploy inference API with monitoring

Module 4: Building AI Agents

1.5 weeks (24 hours) 9 live + 15 self-paced

Core Topics

  • Pydantic AI fundamentals: type-safe agent design
  • Single-agent patterns: ReAct loops with memory
  • Multi-agent orchestration: handoff and dependencies
  • Production deployment: tracing, safety, monitoring

Hands-on Labs

  • Build support agent with 3 tools (search, escalate, summarize)
  • Create 2-agent system with handoff patterns
  • Deploy agent API with concurrent request handling
  • Multi-agent IT helpdesk: triage → resolve → notify
